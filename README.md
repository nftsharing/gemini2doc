# Gemini to DOCX

[English](#english) | [中文](#chinese)

---

<h2 id="english">English</h2>

**Gemini to DOCX** is a Chrome extension that allows you to easily export your conversations from [Gemini (gemini.google.com)](https://gemini.google.com) into beautifully formatted `.docx` documents. 

### ✨ Features
- **Complete Conversation Export**: Automatically scrolls through the page to capture all conversation turns (User Prompts + AI Responses).
- **Format Preservation**: Retains the original formatting of the conversation.
- **Citation Control**: Options to keep or remove citations from the AI's responses.
- **User-Friendly UI**: A clean, modern popup interface to select the active Gemini tab and export with one click.

### 📦 Installation
Currently, the extension can be installed in developer mode:
1. Open Google Chrome and navigate to `chrome://extensions/`.
2. Enable **"Developer mode"** in the top right corner.
3. Click **"Load unpacked"**.
4. Select the folder containing the extension files.
5. The extension is now installed and ready to use! You can turn off "Developer mode" if you wish.

### 🚀 Usage
1. Open a conversation on [Gemini (gemini.google.com)](https://gemini.google.com).
2. Ensure the conversation has fully loaded.
3. Click the extension icon in your browser toolbar (or select `Gemini to DOCX` from the extensions menu).
4. In the popup, select the tab containing your Gemini conversation (marked with a ⭐).
5. Click **"📄 Export DOCX"**.
6. Choose where to save your file, and you're done!

### 🔄 Updates in V2.2
- Completely refactored and migrated support from `aistudio.google.com` to `gemini.google.com`.
- Improved auto-scrolling and element-capturing mechanisms for the new Gemini UI.
- UI improvements and general stability fixes.
- Open-sourced!

<h2 id="chinese">中文</h2>

**Gemini to DOCX** 是一款 Chrome 浏览器扩展程序，能够将您在 [Gemini (gemini.google.com)](https://gemini.google.com) 的对话内容完整、快速地导出为格式精美的 `.docx` 文档。

### ✨ 功能特色
- **完整对话导出**：自动滚动页面，抓取所有对话轮次（包含用户的提问和 AI 的回答）。
- **保留排版格式**：支持保留对话的原始格式和结构。
- **引用控制**：可选择是否在导出时去除 AI 回答中的引用标记。
- **现代化 UI**：界面简洁美观，支持一键选择 Gemini 标签页并导出。

### 📦 安装方法
目前可以通过开发者模式安装：
1. 打开 Chrome 浏览器，在地址栏输入：`chrome://extensions/`
2. 打开右上角的 **"开发者模式"** 开关。
3. 点击左上角的 **"加载已解压的扩展程序"**。
4. 选择包含此插件代码的文件夹。
5. 安装完成！此时可以关闭"开发者模式"。

### 🚀 使用说明
1. 打开 [Gemini (gemini.google.com)](https://gemini.google.com) 的对话页面。
2. 确保页面内容和历史对话已加载完毕。
3. 点击浏览器工具栏中的扩展图标（或从扩展菜单中找到 `Gemini to DOCX` 并固定）。
4. 在弹出的窗口中，下拉选择您要导出的标签页（Gemini 标签页会带有 ⭐ 标记）。
5. 点击 **"📄 导出 DOCX"** 按钮。
6. 选择保存位置，即可完成文档导出！

### 🔄 V2.2 版本更新说明
- 全面重构，将主要支持的目标平台从 `aistudio.google.com` 迁移至目前的 `gemini.google.com`。
- 优化了对最新 Gemini UI 界面的自动滚动逻辑和对话内容抓取机制。
- 修复了若干已知问题，提升了导出稳定性和性能。
- 正式开源发布。
